# 🎬 CineXpress

A full-stack movie ticket booking system where users can browse movies, select theatres, choose seats, and book tickets online.

---

## 🚀 Features

* 🎥 Browse movies
* 🏢 View theatres and shows
* 💺 Seat selection system
* 🎟 Ticket booking
* 🔐 Authentication (User / Theatre Owner / Admin)
* 📍 Location-based theatre filtering

---

## 🧱 Tech Stack

**Frontend:**

* React.js

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

---

## 📁 Project Structure

```bash
movie-booking/
  client/   # Frontend (React)
  server/   # Backend (Node.js)
```

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/akhiljoseph684/CineXpress.git
cd CineXpress/movie-booking
```

---

### 2️⃣ Install dependencies

#### Backend

```bash
cd server
npm install
```

#### Frontend

```bash
cd client
npm install
```

---

### 3️⃣ Run the project

```bash
# Backend
cd server
npm run dev

# Frontend
cd client
npm start
```
